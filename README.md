# popopy

Install python from Python.org
AND
Install python from Windows store


Clone this project

In the project folder run: 

- pip install -r requirements.txt
-  & ./env/Scripts/Activate.ps1
- python main.py

Open:

http://localhost:5000/vacancies


Operations:

- Get /Vacancies
- Post /Vacancies

- Get /Vacancies/:id
- Post /Vacancies/:id
- Patch /Vacancies/:id
- Delete /Vacancies/:id
